# How to Run JavaScript Program

### 1. Using Browser

* Create `.html` file
* Add script tag:

```html
<script src="script.js"></script>
```

* Open in browser

---

### 2. Using Node.js

Install Node.js

Check version:

```
node -v
```

Run file:

```
node filename.js
```
